## Key Features Summary

### Home Row Mods
- **Hold** key for modifier, **Tap** for letter
- **macOS Layer 0:** H=Ctrl, I=Alt, E=Cmd | T=Cmd, R=Alt, N=Ctrl
- **Windows Layer 1:** H=Win, I=Alt, E=Ctrl | T=Ctrl, R=Alt, N=Win
- **Tapping Term:** 200ms

### OS-Agnostic Operations
All clipboard and navigation shortcuts automatically adapt to your active base layer:
- **macOS (Layer 0):** Uses Cmd (⌘) modifier
- **Windows (Layer 1):** Uses Ctrl modifier

### Layer Access Methods
- **MO(layer):** Hold key to activate layer, release to return
- **TO(layer):** Toggle to layer (stays until switched)
- **OSL(layer):** One-shot layer (active for next keypress only)
- **LT(layer, key):** Hold for layer, tap for key

### Special Keys
- **CW_TOGG:** Caps Word (capitalizes until space/punctuation)
- **QK_REPEAT_KEY:** Repeats last key pressed
- **QK_LEAD:** Leader key for special sequences
- **OSM(MOD_LSFT):** One-shot shift (active for next keypress)

### German Layout Support
- Umlauts: Ä, Ö, Ü, ß
- Currency: €
- All special characters follow German keyboard layout (DE_*)
